## Overview

The objective of this project was to analyze NFL officiating data to identify trends in penalties, replay reviews, and officiating decisions across multiple seasons.

---

## Research Questions

- How have penalty trends changed over time?
- Which penalties occur most frequently?
- Which teams receive the most penalties?
- Are replay review outcomes changing?
- What patterns exist across officials and seasons?
